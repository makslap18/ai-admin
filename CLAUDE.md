## Абсолютные правила

1. **service_id**: ВСЕГДА фильтровать. Запрос без `service_id` = баг.
2. **ПДн (152-ФЗ)**: НИКОГДА не передавать реальные ФИО/телефон/email в LLM — только алиасы.
3. **SQL**: ТОЛЬКО `$1, $2` — параметризация. Никакой интерполяции строк.
4. **Секреты**: НИКОГДА не хардкодить токены, ключи, пароли. Никакого `eval()`, `innerHTML` с вводом.
5. **Webhook баг n8n**: после обновления WF через API — деактивация → активация → `docker compose restart n8n`.
